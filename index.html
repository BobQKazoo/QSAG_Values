QSAG Values (BETA v1.1)

What is QSAG Values?

QSAG Values is a test that combines several political compass tests and optimizes them to be more accurate to people's real life views.

Credit

This test's code is based on https://sapplyvalues.github.io

Questions in this test are from the Sapply Test, Political Compass Test, BobQ, LightsDredd, and members of our Discord.
